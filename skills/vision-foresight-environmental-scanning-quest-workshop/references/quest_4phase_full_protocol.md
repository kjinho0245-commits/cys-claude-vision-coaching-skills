# QUEST 4-Phase Full Protocol — Facilitation Script + 박사님 3 컨텍스트 사례

## 전체 일정 표준

| Phase | Duration | Activity |
|---|---|---|
| Phase 1 (Preparation) | 4~6 weeks 사전 | 4 tasks |
| Phase 2 (Scanning Workshop) | 1 day (9am~6pm) | All-day workshop |
| Phase 3 (Analysis & Report) | 2 weeks 사후 | Report 작성·배포 |
| Phase 4 (Strategic Options) | Half day (9am~13pm) | Half-day workshop |

총 6~8 weeks, 1.5 days workshop time.

## Phase 1 — Preparation 풀 protocol

### Task 1.1 — Strategic Issue Definition

#### Facilitation 워크북 (사전 박사님 작성)

```markdown
# QUEST Strategic Issue Definition Worksheet

## Q1: Issue 한 문장으로 진술
[1 sentence]

- [ ] Yes (5y 이상 미래)
- [ ] No → re-frame

## Q3: 조직 전략 차원 영향?
- [ ] Yes (mission/objectives 영향)
- [ ] No → 더 큰 issue로

## Q4: Time horizon
- [ ] 5y
- [ ] 10y
- [ ] 15~20y

## Q5: 다층 stakeholder 영향?
- 영향 받는 stakeholders 5~10:
  1. ...
  2. ...

## Q6: 시나리오 분기 가능?
- [ ] Yes (여러 미래 시나리오 가능)
- [ ] No → 다른 issue 선정

## Q7: Final issue statement
[refined 1~2 sentences]
```

## Phase 2 — All-day Workshop 풀 protocol

### Workshop 시간표 (9am~6pm)

#### 9:00~9:30 — Opening
- 박사님 환영·QUEST 소개·룰 (mutual respect·active participation·anonymity in scoring·confidentiality)
- 일정 안내
- 참가자 self-intro (1분씩)

#### 9:30~11:00 — Mission/Purposes/Objectives 토의

**Facilitation script**:
1. (10분) 현행 mission statement display·읽기
2. (15분) 각자 *우리 institution의 진짜 mission*은 뭔가? sticky notes
3. (20분) 그룹 share·논의
4. (20분) Refine·합의
5. (15분) Purposes 3~5개 명료화
6. (10분) 5~10y objectives

**Output 1**: Refined mission·purposes·objectives

#### 11:00~11:15 — 휴식

#### 11:15~12:30 — Open-ended Trends 토의

**Facilitation script**:
1. (15분) Brainstorming — 각자 *미래에 큰 영향 줄 trends·events* 5~10개 sticky notes
2. (15분) Wall에 모두 붙이기
3. (20분) 그룹 cluster·중복 제거
4. (15분) 10 도메인 분류 (Gordon & Glenn 9개 + Spirituality — SKILL.md 도메인 목록 기준 엄수)
5. (10분) Top 20~30 trends 우선순위

**Output 2**: 30~50 trends list, domain classified

#### 12:30~13:30 — 점심

#### 13:30~15:30 — Cross-impact Analysis

**Facilitation script**:
1. (15분) 매트릭스 구조 설명 + sample
2. (45분) Pairs·triplets로 분담 — 각 trend pair에 cross-impact score
   - +3: strong positive
   - +2: moderate positive
   - +1: weak positive
   - 0: neutral
   - -1: weak negative
   - -2: moderate negative
   - -3: strong negative
3. (30분) 매트릭스 종합·discussion
4. (30분) Top *most influential* trends + Top *most affected* trends 식별

**Output 3**: Cross-impact matrix, top trends

#### 15:30~15:45 — 휴식

#### 15:45~17:30 — Alternative Scenarios Sketch

**Facilitation script**:
1. (20분) 4가지 미래 가능성 framework 소개 (vision-four-futures)
2. (60분) Small groups (3~4명씩 4 그룹) — 각 그룹이 시나리오 1개 sketch
3. (25분) 각 그룹 발표·논의

**Output 4**: 4 시나리오 sketch (각 1 페이지)

#### 17:30~18:00 — Wrap-up

**Facilitation script**:
1. Phase 3 일정 안내 (report 작성·배포)
2. Phase 4 일정 합의
3. Pre-Phase 4 task: report review·comments
4. Closing remarks

## Phase 3 — Report 작성 풀 protocol

### Report 작성 timeline

| Day | Activity |
|---|---|
| Day 1~3 | Workshop output 정리·organize |
| Day 4~7 | Part 1 (Mission·Stakeholders) 작성 |
| Day 8~12 | Part 2 (Scenarios) 작성 |
| Day 13~14 | Review·revision |
| Day 15 | 참가자에게 배포 |
| Day 15~21 | 참가자 comments 수렴 |
| Day 22 | Final report 확정 |

### Report Part 2 — 시나리오 작성 protocol

각 시나리오마다:

#### Step 1 — Defining Characteristics (1 단락)
- 핵심 한 줄
- 정의적 trends 3~5개
- 시나리오 *분위기·logic*

#### Step 2 — Key Trends·Events (5~10 bullets)
- 시나리오를 만드는 핵심 trends
- 확률·영향 표시

#### Step 3 — Implications for Institution (10~15 bullets)
- Institution의 *opportunities*
- Institution의 *threats*
- Strategy 함의

#### Step 4 — Trigger Events (5~7 bullets)
- 시나리오가 *현실화 신호*
- 모니터링할 leading indicators

## Phase 4 — Half-day Workshop 풀 protocol

### Workshop 시간표 (9am~13pm)

#### 9:00~9:30 — Opening + Phase 3 Report 요약

**Facilitation script**:
1. 박사님 인사
2. Phase 3 report 핵심 요약 (15분)
3. Phase 4 목적·룰 안내

#### 9:30~11:00 — Strategic Options 평가

**Facilitation script**:
1. (15분) 시나리오별 implications에서 Options 도출 (브레인스톰)
2. (30분) Options 10~20 list·정제
3. (30분) Probability-Impact + Strategic Fit 평가
4. (15분) Tier 1 (즉각)·Tier 2 (검토)·Tier 3 (모니터링) 분류

**Output 5**: 평가된 Strategic Options

#### 11:00~11:15 — 휴식

#### 11:15~12:30 — Strategic Issues 시리즈 정제

**Facilitation script**:
1. (10분) PDF 핵심 명제 재확인 — *QUEST는 정책 결정 X, strategic issues identification만*
2. (30분) Options에서 *추가 연구·결정 필요한 strategic issues* 추출 (5~10개)
3. (20분) 각 issue를 *future research question*으로 변환
4. (15분) 우선순위·시급도 평가

**Output 6**: Top 5~10 Strategic Issues 시리즈

#### 12:30~13:00 — Follow-up + Wrap-up

**Facilitation script**:
1. (15분) Next steps — 누가 무엇을 언제까지
2. (10분) Renfro 4-stage cycle 통합 — strategic issues가 Stage 1·2·3 입력
3. (5분) 다음 QUEST cycle 일정 (보통 연 1회)
