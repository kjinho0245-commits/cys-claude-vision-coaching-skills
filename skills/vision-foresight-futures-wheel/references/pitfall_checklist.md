# 미래 예측 함정 체크리스트 — 흔히 빠지는 18가지

미래 예측은 본질적으로 빗나가기 쉽다. 그러나 빗나가는 패턴은 *반복*된다. 각 패턴을 알면 분석 품질을 80%까지 끌어올릴 수 있다.

본 체크리스트는 분석 산출 직전 *자가 점검 도구*다.

---

## A. 인지 편향 함정 (Cognitive Biases)

### 1. 직선 외삽 함정 (Linear Extrapolation Fallacy)
"지금 이 트렌드가 그대로 10년 가면..." → S곡선·임계점·역전 무시.

✓ 점검: 트렌드가 무한 지속한다고 가정하지 않았는가? 한계·포화점·역전 신호를 명시했는가?

### 2. 가용성 휴리스틱 (Availability Heuristic)
최근 뉴스에서 본 사건을 과대평가 (예: 코로나 직후 모든 분석이 "팬데믹 빈발"로 쏠림).

✓ 점검: 분석이 *최근 6개월 뉴스*에 과도하게 의존하지 않는가?

### 3. 확증 편향 (Confirmation Bias)
"내가 옳다고 생각한 시나리오를 뒷받침하는 증거만" → 반대 신호 무시.

✓ 점검: 내 결론과 *반대 방향*의 증거를 의식적으로 한 번 찾아봤는가?

### 4. 권위·합의 편향
"세계경제포럼이 그렇다고 말했으니..." → 권위 의존, 독자적 추론 결여.

✓ 점검: 전문가 합의를 *복기*만 했는가, 아니면 *재검토*했는가?

### 5. 정상성 편향 (Normalcy Bias)
"이번에도 그럭저럭 넘어갈 것" → 검은 백조·구조적 위기 과소평가.

✓ 점검: "이번엔 다르다" 시나리오를 한 번이라도 진지하게 다뤘는가?

### 6. 단기 과대·장기 과소 평가
인간은 단기 변화를 과대평가, 장기 변화를 과소평가 (Amara의 법칙).

✓ 점검: T+1~3년 영향이 실제보다 부풀려지지 않았는가? T+15~30년이 너무 평탄하지 않은가?

## B. 분석 구조 함정 (Analytical Pitfalls)

### 7. 차원 쏠림 (Dimension Imbalance)
STEEPS 6차원 중 1~2개만 두꺼이, 나머지는 빈약.

✓ 점검: 각 차원에 최소 1개 영향이 있는가? 영성(Sp)이 누락되지 않았는가?

### 8. 단조 부호 (Monotonic Sign)
모든 영향이 🟢이거나 모두 🔴.

✓ 점검: 🟢↔🔴 반전이 최소 1회 이상 명시되었는가? (세옹지마 원칙)

### 9. 단순 강도 증가 (Strength Amplification)
1차→2차→3차가 *질적*으로 다르지 않고 *강도만* 증가.

✓ 점검: 각 차수에서 *새로운 메커니즘·새 영역·새 대상*이 등장하는가?

### 10. 시간 척도 압축 (Time Scale Compression)
1차·2차·3차가 다 5년 이내에 발생한다고 잘못 설정.

✓ 점검: 각 차수가 5년 이상 시간 간극을 두는가?

### 11. 동어 반복 (Tautology)
"AI가 자동화를 가속화한다" 식 *입력=출력* 진술.

✓ 점검: 각 영향이 입력과 *다른 무엇인가*를 가리키는가?

### 12. 추상도 혼선 (Mixing Abstraction Levels)
한 차수 안에 "사회 양극화 심화" 같은 거대 추상과 "신입 SI 채용 50% 감소" 같은 구체가 뒤섞임.

✓ 점검: 한 표 안에서 추상도 일관성을 유지했는가?

## C. 사실·데이터 함정 (Factual Pitfalls)

### 13. 할루시네이션 (Hallucination)
존재하지 않는 통계·논문·인물 인용.

✓ 점검: 모든 통계·인물·사건이 검증 가능한가? 불확실한 것은 "추정" 표시되어 있는가?

### 14. 출처 누락
"한 연구에 따르면..." 식 막연한 인용.

✓ 점검: 출처가 명시되어 있거나 "출처 미확인—직접 검증 권장" 표시되어 있는가?

### 15. 글로벌-한국 혼동
미국·유럽 분석을 한국에 그대로 적용.

✓ 점검: 한국 특수성(인구·정치·종교·경제 구조)을 명시적으로 반영했는가?

### 16. 자료 구식
2년 전 통계로 현재를 분석.

✓ 점검: 사용한 데이터의 *기준 연도*를 명시했는가? 가능하면 최신 자료로 갱신했는가?

---

## D. 의사결정·서사 함정 (Decision/Narrative Pitfalls)

### 17. 액션 부재 (No-Action Output)
분석은 두꺼우나 의사결정자가 *지금 무엇을 해야 하는지* 불분명.

✓ 점검: "8단계 액션 인사이트"가 *지금 실행 가능*한 형태로 적혔는가?

### 18. 회의주의 도피 (Skeptical Escape)
"미래는 알 수 없다, 다 변할 수 있다" 식 무책임한 모호함.

✓ 점검: 어느 시점에 어떤 시나리오가 어떤 확률로 가능성 있는지 *입장*을 적었는가?

## 점검 순서

산출 직전 다음 순서로 점검:

1. **차원 점검** (#7, #15) — STEEPS 균형, 한국 맥락
2. **부호 점검** (#8) — 세옹지마 반전 명시
3. **차수 점검** (#9, #10, #11) — 질적 변화·시간 간극·동어 반복 회피
4. **사실 점검** (#13, #14, #16) — 할루시네이션·출처·자료 신선도
5. **편향 점검** (#1~#6) — 직선 외삽·가용성·확증·권위·정상성·시간 비대칭
6. **출력 점검** (#12, #17, #18) — 추상도·액션·회피 회의주의

각 항목 ✗ 발견 시 보강 후 산출.

- 이 분석이 빗나갈 가장 큰 가정 1~2개

- 후속 추적 지표 (분석을 검증·갱신하기 위해 봐야 할 신호)
