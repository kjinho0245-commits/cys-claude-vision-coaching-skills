## Decision Tree

```
Top issue 등장
  │
  ▼
Q1: 의사결정 *명확*한가? (어떻게 대응할지 알고 있나?)
  │
  ├─ Yes → Q2: 시급한가? (수개월 내 행동 필요?)
  │           │
  │           ├─ Yes → Option 1: Recommendation (즉시 권고·실행)
  │           │
  │           └─ No → Q2.1: 복잡도 高?
  │                    │
  │                    ├─ Yes → Option 2: Development Task Force (전략 정밀 설계)
  │                    │
  │                    └─ No → Option 1: Recommendation (점진 실행)
  │
  └─ No → Q3: 정보·연구 부족인가?
              │
              ├─ Yes → Option 3: Research Task Force (외부·내부 deep research)
              │
              └─ No → Q4: Wild card (low prob × high impact)?
                          │
                          ├─ Yes → Option 4: Directed Scanning (모니터링 강화)
                          │
                          └─ No → Q5: Insufficient Consensus?
                                      │
                                      ├─ Yes → Stage 2 Recycle
                                      │
                                      └─ No → 재평가 필요
```

## Option 1 — Issue Strategy Recommendation

### 적용 조건
- Tier 1 (Strategic) issue
- 의사결정 *명확*: 어떻게 대응할지 알고 있음
- 시급도: high (수개월 내 행동)
- 복잡도: low~medium

### 산출
- *Recommendation document* (1~3 페이지)
- Decision-maker (CEO·당회·박사님)에게 직접 제출
- 즉각 실행 plan 포함

### 박사님 컨텍스트 적용

#### 미래학·연구소
- 박사님 신년 단행본 출간 핵심 메시지 결정
- 자문 활동 priority 결정
- 미래학 강의 시즌 priority 결정

#### 목회·교회
- 당회 권고 → 사역 결정 (특정 사역·예산 배정)
- 신년 비전 선언 (1월 시무식)
- 청년 사역 paradigm shift 결정

#### 금융투자
- Portfolio 즉각 조정 (특정 종목 매수·매도)
- Asset allocation 변경

### 함정
- *Confirmation bias* — 박사님이 *원하는* 결정을 권고로 위장 가능
- 대응: 적어도 1개 *반대 시나리오* 명시 + Devil's advocate 코멘트

## Option 2 — Issue Strategy Development Task Force

### 적용 조건
- Tier 1 (Strategic) 또는 T2 (Action) issue
- 의사결정 명확하나 *복잡도 高* (다층·다부서·다이해관계자)
- 시급도: medium (6~12개월)
- 자원 투입 大

### 산출
- *Strategy paper* (10~30 페이지)
- 1년 implementation roadmap
- KPI·milestones 정의
- Budget·resource plan

### Task Force 구성 권고
- Lead: 박사님 또는 박사님 위임 사람
- Members: 분야별 전문가 5~10명 + 외부 자문 2~3명
- Cycle: 월 1회 회의 + 분기 보고
- Duration: 6~12개월

### 박사님 컨텍스트 적용

#### 미래학·연구소
- *대형 단행본·시리즈* 기획 (예: AGI 5권 시리즈)
  - Lead: 박사님
  - Members: 자문위원 + 외부 expert essays
  - Cycle: 월 진행 review
- *대형 자문 프로젝트* (정부·대기업 1년 자문)

#### 목회·교회
- *교회 5년 비전 plan* 수립 위원회
  - Lead: 박사님 (담임)
  - Members: 부목사·당회·평신도 위원
- *청년 사역 paradigm shift* TF

#### 금융투자
- *포트폴리오 5년 재편 plan* (시나리오별)

### 함정
- *Task Force 무한 회의* — 결정 지연
- 대응: 명확한 timeline + final deliverable 정의 + interim milestones

## Option 3 — Issue Research Task Force

### 적용 조건
- 의사결정 *불명확* (정보·연구 부족)
- *추가 deep research* 필요
- 시급도: low~medium (6~18개월)

### 산출
- *Research report* (50~100 페이지)
- 의사결정 framework·옵션 평가
- *다음 cycle Stage 3* 입력

### Research 방법
- Literature review (vision-foresight-environmental-scanning-techniques 2번 기법)
- Expert panel·1:1 인터뷰 (1번 기법)
- Modeling·시뮬레이션 (필요 시)
- Scenario analysis (vision-foresight-futures-wheel)
- 외부 thinktank·대학 협력

### 박사님 컨텍스트 적용

#### 미래학·연구소
- *새 분야 진입 research* — 박사님 미지 영역 (예: 양자컴퓨팅·합성생물학 깊이 진입 시)
  - 외부 expert 1:1 인터뷰 5~10명
  - 학술 논문 review 50편
  - 6~12개월
- *새 단행본 분야 research*

#### 목회·교회
- *한국 교회 미래 분야* deep research (3년 단위)
- 청년 사역 *국제 비교* research

#### 금융투자
- *새 자산군 진입* research (예: 디지털 자산·private equity 등)

### 함정
- *Analysis paralysis* — research 무한 확장
- 대응: 명확한 research questions + 결정 deadline

## Option 4 — Directed Issue Scanning Task Force

### 적용 조건
- Wild card issue (low prob × high impact)
- Watch closely issues
- 또는 *지속 모니터링* 강화 필요

### 산출
- 분기·월간 *모니터링 보고서*
- Trigger event 알림
- Indicator dashboard

### Scanning Team 구성
- Dedicated scanner 1~2명 (해당 issue 전담)
- Specific keywords·sources·indicators
- Frequency: 주간 또는 일간 (issue 시급도에 따라)

### 박사님 컨텍스트 적용

#### 미래학·연구소
- *AGI capability monitoring* — 매주
  - KMMLU·HumanEval·GPQA 점수
  - OpenAI·Anthropic·Google blog
  - 한국 LLM 평가
- *한반도 통일 indicators* — 분기
- *글로벌 지정학* indicators — 월간

#### 목회·교회
- *한국 기독교 인구* 지표 — 분기·연간 (한목협·통계청 일정 sync)
- *청년 사역 indicators* — 분기

#### 금융투자
- *경제 indicators* — 일간·주간
- *시장 indicators* — 실시간

### 함정
- *모니터링 fatigue* — 너무 많은 indicator·alert
- 대응: Tier 1 (매일·매주) vs Tier 2 (분기) vs Tier 3 (연 1회) 구분

## 박사님 통합 권고 — 컨텍스트별 typical mix

### 아시아미래인재연구소 — 분기 통합 권고

분기 Top 5 issues 분배:
- 2 issues → Option 1 Recommendation (단행본·강의 즉시 메시지)
- 1 issue → Option 2 Development TF (대형 프로젝트)
- 1 issue → Option 3 Research TF (deep dive)
- 1 issue → Option 4 Directed Scanning (Wild card)


연 Top 3-5 issues 분배:
- 1 issue → Option 1 Recommendation (당회 → 신년 비전)
- 1 issue → Option 2 Development TF (사역 위원회)
- 1~2 issues → Option 3 Research (외부 자문·박사님 deep dive)
- 1 issue → Option 4 Directed Scanning

### 금융투자 — 분기·월 cycle

분기 Top 3-5 issues:
- 2~3 issues → Option 1 Recommendation (포트폴리오 즉시)
- 1 issue → Option 4 Directed Scanning (시장 변동성·indicators)

## Strategy 산출 → 다음 cycle Feedback

각 strategy 실행 후 *결과를 다음 cycle Stage 1*에 환류:
- Recommendation 실행 결과 → 평가
- Development TF 산출물 → 신규 issues 발견
- Research TF 결과 → Stage 3 재평가
- Directed Scanning → 신규 records → Stage 1 재투입

→ 이 환류가 시스템 *학습*의 핵심 (Gordon-Glenn Generic Futures Scanning System의 *Feedback & New Requirements*).
