# Figure 5: VCR in Conscious Technology Scenario

> **출처**: Glenn, J. C. (2009). Futures Wheel. In J. C. Glenn & T. J. Gordon (Eds.), *Futures Research Methodology — Version 3.0* (Chapter 6, §III.C, Figure 5). Washington, DC: The Millennium Project.

## 시나리오 배경

- **시나리오명**: Conscious Technology (Post-Information Age)
- **핵심 전제**: *"The Post-Information Age in which distinctions between technology and consciousness blur"* (Glenn 2009, §III.C)
- **Item**: VCR (Videocassette Recorder)
- **변형 프레이밍**: *"One could imagine that the VCR is a conscious entity capable of communicating"* (Glenn 2009, §III.C)

## PDF Figure 5 재현 — 14 Features

Glenn (2009) §III.C Figure 5에서 명시된 features (원문 기재 순서):

| # | Feature | 설명 | 연결 가정 |
|---|---------|------|-----------|
| 1 | voice activation | 음성으로 VCR에 명령 | Voice/gesture interface |
| 2 | microphone | voice activation 구현 하드웨어 | Voice/gesture interface |
| 3 | voice reproduction program | 음성 인식 소프트웨어 | Voice/gesture interface |
| 4 | built-in modem | 네트워크 연결 | Computing power |
| 5 | search TV schedules | TV 프로그램 자동 검색 | Computing power |
| 6 | remote visual data banks | 원격 영상 데이터베이스 접근 | Computing power |
| 7 | electronic funds transfer | 전자 결제 기능 | Computing power |
| 8 | viewing pattern analysis | 시청 패턴 분석 | Computing power |
| 9 | feedback on previous patterns | 이전 시청 패턴 피드백 | Computing power |
| 10 | computer memory of interests | 관심사 컴퓨터 저장 | Computing power |
| 11 | on-line access | 온라인 네트워크 접근 | Computing power |
| 12 | analysis computer program | 분석 컴퓨터 프로그램 | Computing power |
| 13 | telelink to visuals | 시각 자료 원격 연결 | Voice/gesture interface |
| 14 | automatically displayed | 자동 표시 기능 | Computing power |

**총 14 features** — Glenn (2009) §III.C Figure 5 기준.

## 검증 주석

- "Automatically Displayed" (표시됨): PDF §III.C에서 "automatically displayed"로 표기. "Displaced"(이동됨)는 오표기.
- "Viewing Pattern Analysis": PDF feature 목록에 포함됨 (#8). wheel에서 반드시 표현해야 함.
- "Data Base": PDF 원전에 별도 항목으로 언급 여부 불명확. 별도 확인 필요 시 원전 직접 참조.
- "Gesture"·"Thought-link": PDF §III.C에 없음. 시나리오 확장으로만 허용, 출처 없음 레이블 필수.
